# GroceryApp — OceanX Assignment

A mini grocery delivery app (Blinkit style) built with Kotlin + MVVM.

## Features
- Login with mobile number + OTP (fake OTP: 1234)
- Home screen with product grid, category filter and search
- Add to cart functionality
- Cart screen with qty controls and bill summary
- Checkout with address input and COD/Online payment
- Order success screen with order ID and delivery ETA

## Tech Stack
- Kotlin
- XML Layouts
- RecyclerView + ViewBinding
- MVVM (ViewModel + LiveData)
- Glide for images

## How to Run
1. Clone the repo
2. Open in Android Studio
3. Sync Gradle
4. Run on emulator or device (API 24+)
5. Login with any 10 digit number
6. OTP is 1234
